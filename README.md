# DoAnTotNgiep

# To install and run.

1. Backend website
+ npm install
+ npm start

2. Backend chatbot
+ pip install requirements.txt
+ python training.py
+ python manage.py

3. Frontend website
+ npm install
+ npm start


4. Frontend mobile
+ yarn install
+ yarn react-native run-android


Trong tương lai gần sẽ tái cấu trúc frontend.
